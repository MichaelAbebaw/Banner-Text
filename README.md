# Banner Text
Writes a stylish text to terminal. 

# Supported Characters  
- [x] a - z  
- [ ] 0-9  
- [ ] especial characters (e.g. !, " ...)  

## Command
`Badge <text> [symbol]` 

### Examples
`Badge "hello there"`    
`Badge eight 8`  
`Badge "My name is Earl" "&"`
