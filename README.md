# Banner Text
Writes a stylish text to terminal.  


## Command
`Badge <text> [symbol]` 
> Only supports characters from a-z.  

### Examples
`Badge "hello there"`    
`Badge eight 8`  
`Badge "My name is Earl" "&"`
